# 2FA

A command-line generator of Time-based Ont-time Password (TOTP) for Two-Factor Authentication (2FA) written in Python3. Reduced from [py2fa](https://github.com/cpiehl/py2fa) which offers a Graphical (Gtk) user interface.


```bash
cp 2fa.json.sample 2fa.json
# Edit to fit your needs; the json format is self-explainatory.
# Then run
./2fa.py
# For options
./2fa.py -h
```
